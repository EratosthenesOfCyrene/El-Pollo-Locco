(static) draw()

This function draws on the canvas all the elemnts needed for the game. Steps: 1. Clears the canvas before redrawing. 2. Translates camera view to follow the character. 3. Renders background, clouds, status bars, game entities, and UI elements. Uses: - this.ctx.clearRect - Clears the entire canvas before redrawing. - this.ctx.translate - Shifts the canvas view horizontally to simulate camera movement. - level.backgroundObjects - Array of background elements to render behind everything else. - level.clouds - Array of cloud objects floating in the background layer. - this.statusBar - Status bar displaying the player's health or energy. - this.statusBarBottles - Status bar showing the number of collected bottles. - this.statusBarCoins - Status bar showing the number of collected coins. - this.statusKilledEnemies - Status bar showing how many enemies were killed. - this.setTextStyle - Applies font and color styling for canvas text. - this.ctx.fillText - Draws the killed enemies number as text on the canvas. - this.resetTextStyle - Resets the canvas text style to default settings. - this.screens - Overlay screens such as start, pause, or game over. - this.character - The player character object rendered in the foreground. - level.enemies - Array of enemy objects the player can interact with. - level.bottleOnFloor - Bottles that the player can pick up from the ground. - level.coins - Collectible coin objects scattered across the level. - this.collectedThrowableObjects - Array of bottles collected and ready to throw.
