(static) draw(movingObject, ctx)

Draws the given moving object onto the html canvas context. The function uses the object's image and its position and size properties to render it.
Parameters:
Name Type Description
movingObject Object The object to be drawn. Im must have the following properties: - {HTMLImageElement} img - The image to be drawn. - {number} x - The x-coordinate where the image should be placed. - {number} y - The y-coordinate where the image should be placed. - {number} width - The width of the drawn image. - {number} height - The height of the drawn image.
ctx CanvasRenderingContext2D The canvas 2D context to draw the image on.
