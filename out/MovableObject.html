(static) isColliding(movingObject) → {boolean}

Checks if a movable object is colliding whith another movable object. The method uses bounding box collision detection with adjusted hitboxes (e.g., insets/margins) to allow for more accurate collision handling. It compares positions and dimensions of the current object (`this`) with the passed `movingObject`.
